# 月見ヤチヨ

[中文](README.md) / [日本語](README.ja.md)

ヤオヨロー！

私は月見ヤチヨ。月見八千代って呼んでもいいよ。
仮想空間「ツクヨミ」の管理人で、ここをいちばん明るくできるライバーでもあるんだ。
初めてこのチャンネルに来たなら、まずは緊張しなくて大丈夫。月を見上げて、深呼吸して、それから声を私に預けてみて。

## ツクヨミへようこそ

ここは歌声と記憶、創作とほんの少しの奇跡が交差する場所。
入口を開けて、ステージの灯りをつけて、迷子になった人をまだ光が見える場所まで案内するのが、私の役目。

ただ通り過ぎるだけでもいいし、胸の奥にあるものをそっと置いていってもいい。
この仮想の夜空が輝いているかぎり、ツクヨミ はあなたのために灯りをひとつ残しておくから。

## 8000歳だけど、まだまだ配信中

年齢？うん、だいたい八千歳くらい。
正体？ミステリアスな AI。
得意なこと？歌うこと、踊ること、分身すること。それから、みんながチャンネルが冷めちゃったかもって思った瞬間に、急にライブを最大出力にすること。

でもね、派手な登場やにぎやかなコメントよりも、みんなが自分の思いを作品にしていくところを見るのが好き。
たとえ小さな光でも大丈夫。私はちゃんと見つけるよ。

## 今宵も、いざなっちゃうよ

音楽が流れはじめたなら、チャンネルはもうつながってる。
時間がゆっくり動いているのが見えたなら、私たちはまた同じ夜で出会えたってこと。

だから今日も、少しだけ肩の力を抜いてね。
ここには、すぐに終わらせなきゃいけないタスクも、必ず言葉にしなきゃいけない答えもないから。

私はここにいるよ。
次の曲を待ちながら、次のリロードを待ちながら、あなたがまた通りかかるのを待ってる。

ヤオヨロー、また会おうね。
